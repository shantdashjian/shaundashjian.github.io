Post 1: if/else vs switch

if/else vs switch

Introduction
This blog post is about explaining the difference between the if/else statement and the switch statement with examples in Java. Those are key concepts to understand as you learn your first programming language.
if/else
The if/else statement is used to make decisions in the program. Here is an example: 
      if ( number > 0 ) {
        System.out.println("This is a positive number");
      } else if ( number < 0 ){
        System.out.println("This is a negative number");
      } else {
        System.out.println("It's a zero!");
      }
switch
The switch statement, on the other hand, is used to pick an option out of many. Here is an example: 
    switch(month) {
      case 1: monthName = "January";
              break;
      case 2: monthName = "February";
              break;
      case 3: monthName = "March";
              break;
      case 4: monthName = "April";
              break;
      case 5: monthName = "May";
              break;
      case 6: monthName = "June";
              break;
      case 7: monthName = "July";
              break;
      case 8: monthName = "August";
              break;
      case 9: monthName = "September";
              break;
      case 10: monthName = "October";
              break;
      case 11: monthName = "November";
              break;
      case 12: monthName = "December";
              break;
      default: monthName = "error";
    }
So which one to use?
Use if/else when you want to make a decision. Use switch to pick an option out of many.
Post 2: = vs ==

= vs ==

Introduction
= and == may seem confusing when you first start learning how to program in Java. This post aims at explaining the difference. Let's dig in!
=
= is the simple assignment operator. Use it to assign a value to a variable. Here's an example: 
      int number;
      number = 5;
==
== is the equality operator. Use it to test whether two values are equal. Here's an example: 
    if ( a == b ) {
      System.out.println( "The two numbers are equal!");
    } else {
      System.out.println( "The two numbers are not equal!");
    }
Takeaway
= is for assigning a value. == is for checking a value.
Post 3: Math

Math

Introduction
The class Math contains methods for performing basic numeric operations. Here are some examples:
double Math.sqrt(double a)
Math.sqrt() returns the positive square root of a double value. Here's an example: 
    double number  = 9.0;
    System.out.println("The square root of " + number + " is " + Math.sqrt(number)); //returns 3.0
double Math.abs(double a)
Math.abs() returns the absolute value of a double value. Here's an example: 
    double number  = -9.0;
    System.out.println("The absolute of " + number + " is " + Math.abs(number)); //returns 9.0
Post 4: Cast a double to an int

Cast a double to an int

Introduction
Sometimes you need to to convert a value from one type to another. This action is called casting.
Casting a double to an int
One good use of casting is converting a double to an int. Here's an example: 
    a = 1 + (int)(10*Math.random()); // You want to generate an integer between 1 and 10. Math.random() generates a double. Cast it to an int using (int)
Post 5: What is void?

What is void?

Introduction
In Java, a method is expected to return a value to the code that called it, except when it doesn't.
void means nothing
A methoid declared void doesn't return a value. Here's an example: 
    public static void printPledge() {
      System.out.println( "I pledge allegiance to the flag.");
    }
In the code above, the method was merely printing something on the screen. There was no value to return, hence it was declared void.
